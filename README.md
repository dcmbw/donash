Donash (Bridge)
Donash is a web-based strategy bridge building game originally conceptualised and created for my friend and I's design class. Seeing the potential for the game, we eventually re-wrote all the rules of the game and made a proper board and pieces. This is my best attempt at a web-based port of the game so anyone can play. Project will stay open-source effectively forever hopefully.
